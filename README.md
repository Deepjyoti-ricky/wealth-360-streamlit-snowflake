## üöÄ Deployment Options

### üèîÔ∏è Streamlit in Snowflake (Recommended)

**Zero-configuration deployment** - Upload and run immediately:

1. **Upload Application Files**
   ```sql
   -- In Snowsight, create new Streamlit app
   -- Upload streamlit_app.py as main file
   -- Upload environment.yml for package dependencies
   ```

2. **Package Dependencies (environment.yml)**
   ```yaml
   name: sf_env
   channels:
     - snowflake
   dependencies:
     - python=3.11
     - streamlit=1.45.1
     - snowflake-snowpark-python
     - pandas
     - numpy
     - plotly
     - pydeck
   ```

3. **Optional Context Configuration**
   ```toml
   # Snowsight Secrets (optional overrides)
   [SNOWFLAKE]
   DATABASE = "FSI_DEMOS"
   SCHEMA = "WEALTH_360"
   WAREHOUSE = "COMPUTE_WH"
   ROLE = "ACCOUNTADMIN"
   ```

4. **Launch** - Application runs immediately with active session

### üíª Local Development

**Full development environment setup:**

```bash
# 1. Clone Repository
git clone https://github.com/Deepjyoti-ricky/wealth-360-streamlit-snowflake.git

# 2. Environment Setup

# 3. Install Dependencies (Automated)

```

**üö® Troubleshooting Dependencies:**

**For Local Development** - If you encounter `ModuleNotFoundError: No module named 'plotly'`:

```bash
# Option 1: Use automated installer
python install_dependencies.py

# Option 2: Manual installation
pip install --upgrade pip
pip install plotly>=5.24.0 streamlit>=1.37.0 pandas>=2.2.0

# Option 3: Force reinstall
pip uninstall -y plotly && pip install plotly>=5.24.0

# Verify installation
python -c "import plotly; print(f'Plotly {plotly.__version__} installed successfully')"
```

**For Streamlit in Snowflake** - If you encounter import errors:

```yaml
# Ensure environment.yml is uploaded with your app
# The file must be at the same level as streamlit_app.py

name: sf_env
channels:
  - snowflake
dependencies:
  - plotly  # This installs from Snowflake Anaconda Channel
  - pandas
  - numpy
  - pydeck
```

**Common Streamlit in Snowflake Issues:**
- ‚úÖ Upload both `streamlit_app.py` AND `environment.yml`
- ‚úÖ Use `channels: - snowflake` (required)
- ‚úÖ Check package availability in [Snowflake Anaconda Channel](https://repo.anaconda.com/pkgs/snowflake/)
- ‚ùå Don't use pip requirements.txt in Streamlit in Snowflake
